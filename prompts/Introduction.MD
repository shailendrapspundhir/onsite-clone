# OnSite 360

## Overview

**OnSite 360** is a platform that connects **blue-collar workers** with **employers and agencies**.  
It enables security guards, labourers, helpers, cooks, cleaners, errand boys, delivery personnel, and other blue-collar workers to **find work**, while allowing employers to **discover, evaluate, and hire workers efficiently**.

Think of it as a **Naukri / Indeed / Monster-like platform for blue-collar workers**, enhanced with:

- AI-assisted onboarding  
- Real-time language support  
- Urgent hiring workflows  
- Location-based matching  

---

## Actors / Agents

### 1. Worker
A person looking for work, such as:
- Security guard  
- Labourer  
- Cleaner  
- Cook  
- Helper  
- Errand boy  
- Delivery personnel  
- Store helper  
- Dry cleaner, etc.

---

### 2. Employer
An entity or individual looking to hire workers, such as:
- Individuals  
- Shop owners  
- Companies  
- Factories  
- Security agencies  
- Apartment complexes / societies  
- Construction companies  
- Institutions, etc.

### 5. Job
A job listing created by an employer, containing:
- Job title  
- Description  
- Number of hours  
- Work location  
- Number of vacancies  
- Contact email  
- Additional job details  

---

### 6. Urgent Job
A special job listing marked as **urgent**, used for cases such as:
- Replacement for a sick or absent employee  
- Short-notice requirements  

**Optional features:**
- Hourly rate  
- Fixed duration employment

## Features

### Worker Capabilities

A worker can:
- Log in using **mobile number + OTP**
- Create a profile with **AI-guided onboarding**
- Provide personal and professional details:
  - Name  
  - Age  
  - Date of birth  
  - Place of birth  
  - Current address  
  - Permanent address  
  - Photograph  
  - Skills narration  
  - Past employment and experience  
- View, search, and filter recommended jobs by:
  - Location (geo-fencing)  
  - Job type  
  - Employer name  
  - Company  
  - Region / city / state  
- View job details  
  *(Exact address and contact details hidden for security reasons)*
- Apply for jobs  
- Track application status in a list view  
- Withdraw job applications  
- Mark availability as **Active / Inactive**
- Subscribe / unsubscribe to:
  - Job alerts  
  - Application status updates  
- Subscribe / unsubscribe to **Urgent Job alerts** based on chosen location  
- Refer friends via:
  - SMS  
  - WhatsApp  
  - Instagram  
  - Facebook  
  - X (Twitter)  
  - Email  
- Contact support  
- Delete profile from the platform  

---

### Employer Capabilities

An employer can:
- Log in using **mobile number + OTP**
- Create a profile with **AI-guided onboarding**
- Add details about:
  - Company / agency / individual  
- View, search, and filter workers by:
  - Location (geo-fencing)  
  - Worker type  
  - Skills  
  - Region / city / state  
- View worker profiles  
  *(Name, contact number, and exact address hidden for security reasons)*
- Post jobs on the platform  
- Unlock worker profiles using platform credits:
  - Initial free credits  
  - Periodic free credits  
  - Paid credits / packages  
- Access unlocked information:
  - Worker name  
  - Contact number  
  *(Address remains hidden for security)*
- Message workers within the platform  
- View applications received for posted jobs  
- Filter, sort, and review applicants  
- Purchase credit packages to unlock more worker profiles  
- Contact support  
- Delete profile from the platform  

---

## Platform Capabilities

The platform can:
- View, add, edit, and delete:
  - Workers  
  - Employers  
  - Jobs  
- Provide **AI-based and human support** to users  
- Manage a **ticket-based support system**:
  - Ticket creation  
  - Status tracking  
  - Updates from workers, employers, AI, and human agents  
- Push credit offers and packages to relevant employers  
- Send notifications and initiate calls for **Urgent Jobs** to relevant workers
